Question:- Explain the purpose of comments in html and provide an example 
Answer:-  Comments in html serves as non rendered text that provides information about the code for human readers ,they are usful for  documenting your code by making notes and temporarily disabling parts of the code without deleting them, comments  are ignored by browsers and do not effect the rendering of the browser
 Example:-
  <!doctype html>
   <html lan=”eng”>
  <head>
<meta charset=”utf=8”>
</head>
<body>
<!---<p>html defines content and structure---></p>
</body>
</html>
> Above comment is explaining about html structure 
> It is un visible
> If we remove <!--> this it will visible
